# 组件 Component

惯性拖动（Inertial Drag）是指：用户拖动元素释放鼠标后，元素会根据拖动时的速度继续滑动一段距离，最后减速停止，就像在真实世界中有“惯性”一样。

## 代码演示

```tsx
import React from 'react';
import {InertiaBall } from 'interestingWeb';

const InertiaBallDemo = ()=>{
    return <InertiaBall />
}

export default InertiaBallDemo
```
